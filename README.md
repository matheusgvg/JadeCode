# 🎮 Jade's Midnight Adventure

Um jogo de plataforma 2D desenvolvido em **Node.js + HTML5 Canvas** como presente de aniversário para uma amiga muito especial, **Jade**.

Inspirado na atmosfera noturna, em viagens de trem e na estética de músicas dos **Arctic Monkeys**, o jogo acompanha Jade em uma aventura pixelada rumo ao misterioso trem **505**.

---

## 📖 História

Em uma noite estrelada, Jade recebe uma mensagem inesperada:

> "O último trem para 505 partirá em breve."

Agora ela precisa atravessar plataformas, coletar discos de vinil, encontrar aliados pelo caminho e superar obstáculos antes que o trem parta.

## 🚀 Como executar

### Instalar dependências

```bash
npm install express
```

### Iniciar o servidor

```bash
node server.js
```

### Abrir no navegador

```text
http://localhost:3000
```
